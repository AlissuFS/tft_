# GitHub Secrets necessários
# Nome do Secret        -> Valor esperado / instruções

FIREBASE_API_KEY        -> Sua chave de API do Firebase
FIREBASE_AUTH_DOMAIN    -> authDomain do Firebase
FIREBASE_PROJECT_ID     -> ID do projeto Firebase
FIREBASE_STORAGE_BUCKET -> storageBucket do Firebase
FIREBASE_MESSAGING_SENDER_ID -> senderId do Firebase
FIREBASE_APP_ID         -> appId do Firebase

TFT_META_API_KEY        -> Chave para acessar API de meta do TFT
TFT_BACKEND_URL         -> URL do backend Python (FastAPI)
